## Description

QuinAutoRun is a simple tool to toggle auto-run in The Quinfall game.

## Usage

1. Download latest exe release from [here](https://github.com/beetron/QuinAutoRun/releases/download/v.1.0b/QuinAutoRun-v1.0b.zip)
2. Run file as Administrator. (You could minimize the program)
3. You will probably have to install .Net 8.0 runtime [Download](https://download.visualstudio.microsoft.com/download/pr/f1e7ffc8-c278-4339-b460-517420724524/f36bb75b2e86a52338c4d3a90f8dac9b/windowsdesktop-runtime-8.0.12-win-x64.exe)and installã€‚Link is from Microsoft.
4. Launch program again.  Use the `Middle Mouse Button` while in-game to toggle auto-run ON/OFF.

<span style="color:red;">*IMPORTANT*</span>  Use when not in UI/Cursor mode.
